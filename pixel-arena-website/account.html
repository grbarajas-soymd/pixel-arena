Cloud Save

Your game saves locally in your browser. Use cloud save to back up your progress and continue on another device.

  • Upload to Cloud — saves your current browser data to the server
  • Download from Cloud — overwrites your local save with the cloud version
  • Cloud save stores all character slots, gear, followers, and progression
